Introduction
============

Stalker leverages the powerful `SQLAlchemy ORM`_ to facilitate interaction with
databases using the Stalker Object Model (SOM). This tutorial introduces you to
the Stalker Python API and SOM. If you're familiar with SQLAlchemy, you'll find
the transition smooth. Otherwise, SOM offers a user-friendly way to manage
databases.

.. _SQLAlchemy ORM: http://www.sqlalchemy.org/docs/orm/tutorial.html
